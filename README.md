
## Landsat Based Land Surface Temperature (LST) Mapping using MATLAB

## Project Overview
This project demonstrates a complete Remote Sensing & GIS workflow using **MATLAB** to estimate **Land Surface Temperature (LST)** from Landsat thermal bands.

## Objectives
- Process Landsat thermal band data
- Convert DN to Radiance and Brightness Temperature
- Estimate Land Surface Temperature (LST)
- Export results as GeoTIFF-compatible outputs
